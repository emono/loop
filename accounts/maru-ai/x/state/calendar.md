# 投稿カレンダー: まるくん@AI速報 X

速報予定・追いかけ中のテーマ。

## コンテンツ柱（backlog・ネタ切れ防止のローテ）
「速報系(1〜8)＝3日鮮度＋discovery で埋める」「箸休め(9〜10)＝鮮度に依らず出せる」。

- **新モデル速報** — リリースを「料金・ベンチ・位置づけ」で（例: Claude Sonnet 5）
- **料金・コスパ比較** — モデル横断で $/Mトークン、用途別に「安く回すならコレ」
- **論文かみくだき** — 注目研究を「要するに何がすごい？」（一次: arXiv/公式）
- **ベンチ解説** — 新ベンチ/スコアの意味をやさしく（SWE-bench・OSWorld 等）
- **機能アプデ速報** — ChatGPT/Claude/Gemini の新機能「何が変わった」
- **オープンモデル動向** — Llama/Mistral/DeepSeek 等の公開・重み配布
- **資金調達・業界** — 大型ラウンド/買収を規模感（調達額・評価額）で
- **規制・政策** — AI規制の動きを中立に「何が決まった/変わる」
- **用語かみくだき** — MCP・エージェント・RAG 等を1ツイート（箸休め）
- **今週の生成AIまとめ** — 週次でハイライト3〜5本を短く（箸休め）
