# PR 向け voice / トーン: 自社プレスリリース（feeed）

`company.md` の発行主体を、プレスリリース文体に具体化。SNS の maru 系とは別トーン（**公式・簡潔・中立**）。

## トーン規約
- **公式で簡潔**。です/ます調。感情語・煽り・絵文字は使わない。
- **事実ドリブン**: 何を・いつ・いくらで・どこで、を明確に。修飾語より数字。
- **中立・誇張しない**: 「画期的」「革新的」等の主観形容は根拠がある時のみ、控えめに。
- **将来見通しは断定しない**: 「〜を予定」「〜を目指す」「〜する見込み」。

## フォーマット規約
- **タイトル**は約30字・結論先出し（誰が何を）。**サブタイトル**で補足1行。
- **リード文**は冒頭1〜2段落で発表要旨（5W1H）が分かるように。
- 本文は段落＋必要に応じ箇条書き。**数字・価格・日付は正確に**。
- 役員/担当**コメントは鉤括弧で引用**、発言者の役職名を明記。
- 末尾に **会社概要（boilerplate）** と **お問い合わせ先** を必ず入れる。
- 解禁がある場合は**エンバーゴ（解禁日時）**を冒頭に明記。

## NG（review でブロック）
- 根拠なき最上級・比較表現。効能効果の断定。将来の断定。
- 事実と評価の混同。未確定事項（`【要確認】`）の残置。
- 表記ゆれ（製品名・社名）、税込/税抜の不統一。
